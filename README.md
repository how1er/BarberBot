# BarberBot
Суть проекта заключается в создании telegram-бота, с помощью которого пользователи имеют возможность легко и быстро записаться на парикмахерские услуги

### Запуск бота
1. Переходишь в папку с main.py
```
cd <путь к папке>
```
3. Устанавливаешь виртуальное окружение (если уже установлено идешь дальше)
```
pip install virtualenv
```
3. Создаешь новое окружение
```
virtualenv venv
```
4. Активируешь его
```
venv\Scripts\activate.bat
```
5. Устанавливаешь все библиотеки
```
pip install -r requirements.txt
```
5. Запускаешь бота
```
python main.py
```
